## Image Format Guidelines

- **Format**: JPG or PNG
- **Optimization**: Compress images for web (aim for < 500KB per image)
- **Aspect Ratios**: 
  - Hero images: 3:2 or 16:9
  - Side images: 4:3 or 3:2
  - Full-width images: 16:9

## Notes

- All images should be optimized for web use
- Images will be automatically optimized by Next.js Image component
- Ensure images have proper alt text (already set in the page component)
