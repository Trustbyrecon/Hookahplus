## 📋 API Specification

### Request (Route Parameter)
```
GET /api/loyalty/wallet/{accountId}/balance
```

### Request (Query Parameters)
```
GET /api/loyalty/wallet/balance?customerId=CUST-123
GET /api/loyalty/wallet/balance?customerPhone=+1234567890
```

### Response (Success)
```json
{
  "accountId": "acc-123",
  "customerId": "CUST-123",
  "customerPhone": "+1234567890",
  "balanceCents": 5500,
  "totalEarnedCents": 10000,
  "totalRedeemedCents": 4500,
  "lastTransactionAt": "2025-11-03T19:00:00Z",
  "isActive": true,
  "createdAt": "2025-11-01T10:00:00Z",
  "updatedAt": "2025-11-03T19:00:00Z"
}
```

### Response (Error)
```json
{
  "error": "Error message"
}
```

**Status Codes:**
- `200` - Success
- `400` - Bad Request (missing identifier)
- `404` - Not Found (account not found)
- `500` - Internal Server Error

## 📊 Response Fields

**Account Info:**
- `accountId` - Account ID
- `customerId` - Customer reference (CUST-xxx)
- `customerPhone` - Customer phone number

**Balance Info:**
- `balanceCents` - Current balance (uses wallet cache)
- `totalEarnedCents` - Lifetime earned credits
- `totalRedeemedCents` - Lifetime redeemed credits

**Metadata:**
- `lastTransactionAt` - Last transaction timestamp
- `isActive` - Account status
- `createdAt` - Account creation timestamp
- `updatedAt` - Last update timestamp
