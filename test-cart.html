Cart Test Page

Open the browser console and click the "Add" buttons to test cart functionality.

You should see console logs showing:

  • "Adding to cart: {item}" when clicking Add buttons
  • "CartProvider: Adding item: {item}" when items are added
  • "CartToggle: items count: X" showing the cart count
  • "CartDrawer: items: [...]" when opening the cart

Visit: http://localhost:3002
