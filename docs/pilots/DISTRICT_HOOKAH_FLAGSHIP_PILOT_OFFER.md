# District Hookah — Flagship Pilot Offer (3 locations / 30 days → 13 rollout)

This is the proposed commercial structure for District Hookah as the flagship “Revenue Operating System” anchor.

## Pilot scope
- **Duration**: 30 days\n- **Locations**: 3 anchors (casino + surge/outdoor + nightclub)\n- **Default anchors**: Diablo’s Cantina (MGM), The Bullpen DC, Opera Ultra Lounge\n- **Operating model**: Hookah+ runs above POS; POS integration is optional during the pilot (shadow mode is acceptable).

## Pilot objectives (what we must prove)
Pilot success is defined by measurable improvements in 3–5 KPIs (see `DISTRICT_HOOKAH_PILOT_METRICS.md`).\n\nStandard KPI set:\n- throughput/utilization delta\n- average session value + upsell attachment\n- turnaround time reduction\n- labor efficiency proxy\n- drift/reconciliation anomaly rate reduction or improved resolution SLA

## Flagship terms (recommended)
Choose one of these structures depending on negotiation posture.

### Option A — Free pilot (flagship exchange)
- $0 for 30 days across 3 locations\n- Requires:\n  - weekly pilot check-in with decision-maker access\n  - permission to use anonymized metrics in internal reporting\n  - option (not obligation) for co-marketing/case study on success\n\n**Rollout trigger**: if success KPIs are met at week 4, District signs the rollout order for all 13 with a defined start date.

### Option B — Discounted pilot (commitment signal)
- 50% discount for 30 days across 3 locations\n- Applies as credit against first 90 days of rollout if District expands to all 13.

### Option C — Paid pilot with guaranteed rollout discount
- Pilot is paid at a reduced rate.\n- On rollout, District receives a pre-negotiated enterprise discount for the first year.

## Rollout commercial model (13 locations)
This is designed to map to multi-unit reality while keeping pricing simple.

- **Per-location subscription**: $400–$600 / location / month baseline\n- **Enterprise layer** (recommended): analytics + reconciliation + priority support\n- **Estimated ACV range**: $120K–$200K depending on tiering and support footprint

## What District gets (deliverables)
- LaunchPad multi-location provisioning\n- QR operations pack per location\n- staff training + runbooks\n- weekly exec rollup report\n- pilot closeout “Valuation Impact” report\n- optional POS integration plan (Square-first)

## What Hookah+ needs (requirements)
- named sponsor and ops lead\n- table/section capacity estimates\n- staff roster roles (emails optional if they prefer generic accounts)\n- permission to run session ops “above POS”\n- access to basic POS export (optional but recommended)

## Co-marketing / case study (optional, structured)
If District wants flagship positioning benefits:\n- a written testimonial quote (post-pilot)\n- permission to cite “multi-unit concessionaire” success metrics\n- if MGM-sensitive: allow anonymized venue naming if needed (your report can remain internal)

## Risk controls (both sides)
- Pilot is explicitly non-disruptive: Hookah+ can run without POS integration.\n- If venue partner blocks workflow changes, pilot success can be measured on the other two anchors.\n- Security: no secrets shared in docs; credentials are environment-managed.

## Signature checkpoint
Week 4 meeting agenda:\n1. KPI results (baseline vs pilot)\n2. operational friction list + fixes\n3. rollout sequencing for 10 additional venues\n4. commercial close (ACV + term + enterprise support)\n
